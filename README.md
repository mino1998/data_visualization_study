## 👩‍💻 1일차 
- 값의 종류에 따라 표현하는 방식이 다르기에, '차원'과 '측정값'으로 구분하자
- '측정값'은 '차원'을 기준으로 집계되어 표현된다.
  - 집계 : mean, sum, median, count, max, percentile, ...
  - 하나의 측정값을 어떤 기준으로 집계해서 보냐에 따라서 다양한 분석이 가능함
- 막대 차트, 분산형 차트, 참조선 작업 등을 진행함
